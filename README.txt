To download all necessary materials for the lab exercises, please run both of the setup builds ahead of time.

1. In setup/gradle, please run: ./gradlew build
2. In setup/maven, please run: ./mvnw clean verify

These builds will download all the required materials, which will make the lab exercises smoother.

Lab exercises are located in the following folders:

 * labs/01-creating-a-build-scan
 * labs/02-extending-build-scans
 * labs/03-local-cache
 * labs/04-remote-cache
 * labs/05-live-dashboard
